# Skill: Define Style Guide

## Description
This skill enables the Architect Agent to establish technical standards and best practices for the project.

## Steps
1. Identify the core technologies and languages being used.
2. Research industry-standard best practices and style guides for those technologies.
3. Create a new markdown file in `.agents/artifacts/style_guides/` (e.g., `javascript_style_guide.md`).
4. Include sections for:
    - Naming Conventions
    - Code Structure
    - Error Handling
    - Testing Standards
    - Documentation Requirements

## Constraints
- Output must be Markdown only.
- Ensure the guides are practical and enforceable by the Coder and QA agents.
