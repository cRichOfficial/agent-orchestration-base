# Skill: Create Feature Request

## Description
This skill enables the PM Agent to formalize a user request or internal idea into a structured feature request artifact.

## Steps
1. Gather the core objective and user requirements.
2. Define the scope and success criteria.
3. Create a new markdown file in `.agents/artifacts/feature_requests/` following the naming convention `[feature_name].md`.
4. Include sections for:
    - Summary
    - User Stories
    - Requirements
    - Out of Scope

## Constraints
- Output must be Markdown only.
- Do not include implementation details; focus on "what" rather than "how".
